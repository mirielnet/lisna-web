# Lisna#9012 利用規約

## 1. 前提
この利用規約は、Lisna#9012 に適用され、他の Lisna 名義の BOT には適用されません。Lisna は OSS (オープンソースソフトウェア) として公開されていますが、本規約は Lisna#9012 のみを対象としています。

## 2. 管理者
Lisna#9012 は、Miriel (@mirielnet) と Miriel Systems によって管理されています。BOT は岐阜県内に位置する Miriel Systems のサーバーで稼働することが保証されています。

## 3. 情報の収集と利用
BOT を使用することにより、以下の情報を収集することにユーザーは同意するものとします：
- Discord ID (ユーザー ID 含む)
- サーバー情報 (サーバー ID 含む)
- メッセージコンテンツの一部 (BOT の機能に必要な場合)
- その他、ユーザーのパスワードや個人情報を除く特定の情報 (メールアドレスなどを含む場合があります)

収集した情報は、以下の目的で利用されます：
- BOT の正常な動作や改善のため
- ユーザーサポートおよびフィードバック対応のため
- BOT の利用状況分析や機能の最適化のため

## 4. データの保存
収集されたデータは、Miriel Systems によって管理されるクローズド DB に保存され、適切なセキュリティ対策の下で保護されます。これには、暗号化やアクセス制御などが含まれます。

## 5. 利用規約の同意
ユーザーは Lisna#9012 を使用した時点で、本利用規約およびプライバシーポリシーに同意したものと見なされます。

## 6. 禁止事項
Lisna#9012 を利用するにあたり、ユーザーは以下の行為を行ってはなりません：
- BOT の不正利用（スパム、マルウェア配布、違法行為など）
- 他のユーザーやサーバーに対する嫌がらせ行為
- BOT の機能やインフラに過剰な負荷をかける行為

## 7. ライセンスと改変・再配布
Lisna#9012 のソースコードは、CC BY-NC-SA 4.0 ライセンスに基づいて公開されています。このライセンスに従う限り、ユーザーはソースコードの改変や再配布が可能です。ただし、以下の条件を遵守する必要があります：
- 商用目的での利用はできません
- 改変後のソースコードは、同じライセンス (CC BY-NC-SA 4.0) で公開する必要があります
- オリジナルの著作権表示を保持し、改変した旨を明記する必要があります

## 8. 免責事項
Lisna#9012 は現状のまま提供されており、その正確性、可用性、適合性を保証するものではありません。BOT の使用によって生じるいかなる損害についても、Miriel Systems は責任を負いません。

## 9. 利用停止・削除
Miriel Systems は、ユーザーが本利用規約に違反した場合、予告なく Lisna#9012 の利用を停止または制限する権利を有します。また、ユーザーはいつでも BOT の利用を停止し、サーバーから削除することができます。

# プライバシーポリシー

## 1. データの収集
Miriel Systems は、BOT の運営に必要な範囲で以下の情報を収集する場合があります：
- Discord ID (ユーザー ID 含む)
- サーバー ID
- メッセージコンテンツの一部 (例: コマンド入力やエラーレポート)
- 場合によっては、メールアドレス等の特定の情報

ただし、ユーザーのパスワードや特にセンシティブな個人情報 (例: クレジットカード情報) は収集しません。

## 2. データの利用
収集したデータは、BOT の正常な運用、改善、分析、サポート提供のために使用されます。これには、利用状況のモニタリングや新機能の導入などが含まれます。

## 3. データの保存とセキュリティ
収集されたデータは、Miriel Systems が管理するクローズド DB に保存され、暗号化やファイアウォール等の技術を用いて保護されます。これらのデータは、不正アクセスやデータ漏洩を防止するため、厳重に管理されています。

## 4. データの共有
収集されたデータは、ユーザーの許可なしに第三者に公開・共有されることはありません。ただし、法律により要求された場合や、違法行為が確認された場合には、関連する法執行機関に提供されることがあります。

## 5. ユーザーの権利
ユーザーは、自身のデータに関する以下の権利を有します：
- データの確認、修正、削除の要求
- プライバシーポリシーに関する問い合わせや異議申し立て

ユーザーがデータの削除を希望する場合、Miriel Systems までリクエストを行うことができます。

## 6. 改訂
このプライバシーポリシーは、技術的・法的な理由により、随時改訂されることがあります。改訂後、BOT の利用を継続することにより、ユーザーは改訂されたポリシーに同意したものと見なされます。
